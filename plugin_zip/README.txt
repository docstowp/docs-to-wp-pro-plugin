Plugin to interact with the "Docs to WP Pro" Google Docs Editor add-on that allows you to publish SEO-optimized posts with smart internal links from Google Docs.

== Description ==

Docs to WP Pro Google Docs Editor add-on helps you to publish Google Docs to WordPress. 

* Export in seconds (not hours)
* No VAs
* Save 30 mins per WordPress Post
* Similar to WordPress publishing experience

Docs to WP Pro is the ONLY plugin that creates a Gutenberg blocks for the content that is pushed from Google Docs. 

== Important ==
* Install the [Docs to WP Pro](https://workspace.google.com/marketplace/app/docs_to_wp_pro/346830534164) Add on from the Google Workspace Marketplace. It allows you to publish content from Google Docs to your WordPress.

= Features  =
* Allows you to add internal links to WordPress posts by automatically finding potential keywords
* Allows you to create/update any type of WordPress posts, page, or custom posts you've created
* Supports both the Classic editor and the Gutenberg editor
* Automatic image compression
* Automatic image Alt Text generation
* RankMath and Yoast Meta Data support


= How does it work? =

Docs to WP Pro utilizes the WordPress REST API to interact with the WordPress site and publish content from WordPress.

Should you have any questions, please feel free to reach out. Thank you for your cooperation.

== Installation ==

To install this plugin from WordPress Dashboard

1. Go to Plugins menu > Add new
1. Search for Docs to WP Pro
1. Activate the plugin through the 'Plugins' menu in WordPress
1. If you are using any page cache plugin, make sure to purge/clear the cache.

To install this plugin manually

1. Download the Docs to WP Pro Plugin
1. Upload `Docs to WP Pro` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. If you are using any page cache plugin, make sure to purge/clear the cache.
